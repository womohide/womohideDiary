[190419]Captain America: The First Avenger

MCUの五作目、『Captain America: The First Avenger』をみた。マジできつい。仮面ライダーの方がマシだぞこれ。まず敵施設の防衛が紙。戦闘シーンも謎。ライダーとショッカーの戦いを見てるみたい。敵の掘り下げがないから何と戦ってるのかもよくわからない。ヒーローと戦争が絡む映画ならウォッチメンの方が面白かったなぁとか思いながらみてた。これ本当に面白くなるのか?
